# Bank App

Aplicação desenvolvida em **React com TypeScript** que simula a tela inicial de um banco digital. O projeto foi criado com foco em componentização, reutilização de código e testes unitários.

## Tecnologias utilizadas

- React
- TypeScript
- Jest (testes unitários)
- React Scripts


## Funcionalidades

- Componente **Header** com o título da aplicação
- Componente **Button** reutilizável com evento `onClick`
- Componente **Card** contendo o formulário de login
- Função de **boas-vindas** acionada ao clicar no botão
- **Teste unitário** da função de boas-vindas


## Como rodar o projeto

### 1. Clone o repositório
git clone https://github.com/seu-usuario/dio-bank-app.git

### 2. Acesse a pasta do projeto
cd dio-bank-app

### 3. Instale as dependências
npm install

### 4. Execute o projeto
npm start

O projeto será aberto no navegador em:
http://localhost:3000


## Como rodar os testes

npm test
